## About HouseFans

### Installation

1. Clone the repo
2. Using the terminal of your choice -> `Git Clone 'SSH or HTTPS key'`
3. Install the node packages -> `npm i`

### Usage

1. Enter npm run dev in the terminal to run the site in developer mode.
